## インストールの仕方

Rocky Linux
```shell
sudo dnf install -y git
```

## Gitの基本

ローカルリポジトリの作成
```shell
git init
```
<br>

個別にコミットしたい場合
```shell
git add [ファイル名]
```
<br>

すべてのファイルをコミットしたい場合
```shell
git add .
```
